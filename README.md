# SmartSpend — Personal Finance Dashboard

A full-stack web application to track income, expenses, and budgets with a live analytics dashboard.

## Features
- Add and categorize income & expenses
- Live dashboard with monthly trends and budget utilization
- AI-powered insights that automatically detect overspending categories
- Interactive charts for visual financial analysis

## Tech Stack
| Tool | Purpose |
|------|---------|
| Python + Flask | Backend REST API |
| SQLite | Database (aggregations, window functions) |
| JavaScript | Frontend interactivity |
| HTML + CSS | UI layout and styling |

## How to Run
````bash
git clone https://github.com/deepakj01102005-blip/smartspend
cd smartspend
pip install -r requirements.txt
python app.py
````

App runs at `http://localhost:5000`

## Project Structure
````
smartspend/
├── app.py          # Flask app and API routes
├── database.py     # DB setup and queries
├── analytics.py    # Spending insights logic
├── static/         # JS, CSS files
└── templates/      # HTML templates
````
